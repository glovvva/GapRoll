## 4. Her Decision-Making Process

### 4.1 Evaluation Criteria (Ranked)

1. **Legal Safety (10/10 importance):** "Czy to spełnia wymogi Dyrektywy?"
2. **Audit Trail (9/10):** "Czy mogę obronić się przed PIP?"
3. **Ease of Use (8/10):** "Czy zrobię to w 15 minut, nie 3 dni?"
4. **Cost (7/10):** "Czy CEO zatwierdzi ten wydatek?" (99 PLN = instant yes, 499 PLN = needs justification)
5. **Control (9/10):** "Czy mogę edytować, jeśli AI się myli?" (CRITICAL: Manual Override non-negotiable)
6. **Trust (8/10):** "Czy to firma polska/europejska? Czy będą w razie problemu?"

**Anti-Criteria (Things She Doesn't Care About):**
- ❌ Innovation ("AI-powered", "cutting-edge") — suspicious, not reassuring
- ❌ Flexibility ("customizable workflows") — sounds complicated
- ❌ Integrations (unless it's ZUS Platnik, Comarch) — Polish accounting software only

**Stage 1: Problem Awareness (Triggered by deadline panic)**
```
Grażyna hears about EU Directive 2023/970 → June 7, 2026 deadline

```

**Stage 2: Solution Research (Google, LinkedIn, peer recommendations)**
```
Grażyna compares:

**Stage 3: Risk Assessment (Checks trust signals)**
```
Questions she asks herself:

- "Czy mają wsparcie?" → Checks if help center in Polish, phone number visible
```

**Stage 4: Trial (14 days, full Compliance tier access)**
```
Grażyna uploads CSV → sees Art. 16 report in 5 minutes → impressed
Checks EVG scores → tries Manual Override → sees it works → trusts system
Exports PDF → shows to CEO → CEO approves 99 PLN/month
```

**Stage 5: Purchase Decision**
```
Factors that seal the deal:
✅ 99 PLN (affordable, no CEO approval needed)
✅ 15-minute setup (no IT department, no training)

## 8. The "Grażyna Test" (UX Validation)

Before shipping ANY feature, ask:

### 8.1 Comprehension Test
**Question:** "Czy Grażyna by to zrozumiała w 5 sekund?"

**Good Example:**
```
Luka płacowa: 12%
[Info icon]: "Średnia różnica wynagrodzenia kobiet vs mężczyzn. 
Art. 16 ust. 2 lit. a) Dyrektywy UE 2023/970"
```

**Bad Example:**
```
Gender Pay Disparity Coefficient: 1.12 (M/F ratio)
AI Confidence: 87%
```

---

### 8.2 Audit Test
**Question:** "Czy PIP/ZUS mógłby to zakwestionować?"

**Good Example:**
```
EVG Score: 78/100
[Breakdown]:
- Skills: 22/25 (Wymaga JS, Python, AWS — poziom senior)
- Effort: 18/25 (Praca przy komputerze 8h/dziennie)
- Responsibility: 20/25 (Odpowiedzialność za zespół 5 osób)
- Conditions: 18/25 (Biuro, standardowe warunki)
[Podstawa prawna]: Art. 4 Dyrektywy UE 2023/970
```

**Bad Example:**
```
Job Score: 78 (AI-generated, click for details)
```

---

### 8.3 Control Test
**Question:** "Czy Grażyna może przejąć kontrolę, jeśli AI się myli?"

**Good Example:**
```
[EVG Score: 78/100]
❌ Nie zgadzam się. Zmieniam na 72.
[Modal opens]:
- Nowy wynik: 72
- Uzasadnienie: "Stanowisko nie wymaga AWS (zmiana w 2024)"
- [Zapisz] → Audit trail recorded
```

**Bad Example:**
```
AI-powered job evaluation (no manual override)
```
